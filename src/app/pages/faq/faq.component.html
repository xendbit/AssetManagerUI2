Frequently Asked Questions

What Is Nifty Row marketplace?

The Nifty Row marketplace is an NFT marketplace that supports the minting, buying, and selling of NFTs. That means that gas tokens from supported chains fuel all transactions on the Nifty Row platform.

Is there a beginner guide for the Nifty Row marketplace?

Yes. Beginners are advised to use the testnet by following this demo guide: Medium Post

What wallets can I use on Nifty Row marketplace?

You can use the following wallets:
MetaMask
WalletConnect
